To-Do List Web Application

This code creates a simple To-Do List web application using HTML, CSS, and JavaScript. The structure is defined in HTML, where a text input field and a button are provided to add tasks. The tasks are displayed as a list of items in an unordered list. Each task has a "Delete" button that allows the user to remove tasks from the list. The design is styled using CSS for a clean, modern look with a centered container, rounded borders, and hover effects for buttons. JavaScript is used to handle the functionality, adding tasks to the list when the button is clicked and removing them when the "Delete" button is pressed.
